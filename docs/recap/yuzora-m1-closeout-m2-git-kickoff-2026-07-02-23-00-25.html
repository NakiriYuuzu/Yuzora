M2 範圍（spec 摘錄）
  • Rust git service：spawn 系統 git + --porcelain=v2 解析、狀態輪詢/監聽；啟動偵測 git，缺 git 顯示 guided setup。
  • 操作面：status / stage / unstage / discard（需確認）/ commit / branch popup（含 checkout）/ fetch / pull / push。
  • 認證橋接（askpass 模式）：GIT_ASKPASS + SSH_ASKPASS（SSH_ASKPASS_REQUIRE=force）、IPC 驗證呼叫來源、憑證 60 秒記憶體快取不落盤、所有 git 操作帶 timeout。
  • diff viewer：@codemirror/merge；staged/unstaged 分開；file tree 與 Git panel 皆可開。
  • merge conflict 最小 UX：porcelain u 條目 + MERGE_HEAD 等偵測、banner + 檔案清單、editor 內 accept current/incoming/both、明確 abort/continue。
  • dirty buffer external-change conflict flow 完整版（共用 diff UI）。
  • project-wide search：Rust streaming 初篩、結果檔案上限 5,000。
